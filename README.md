#Smart Retail Management System (SRMS)

A SQL Server database project developed to manage retail operations including products, categories, suppliers, warehouses, customers, orders, employees, and payments.

##Features

- Database Design & Normalization
- ER Diagram and Relational Schema
- CRUD Operations (DDL, DML, DQL)
- Joins and Subqueries
- Functions and Stored Procedures
- Triggers and Views
- Indexing and Query Optimization
- AI-Assisted SQL Query Generation

##Database Modules

- Product Management
- Category Management
- Supplier Management
- Warehouse Management
- Customer Management
- Employee Management
- Order Processing
- Payment Management

##Technologies Used

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
- T-SQL

##Key SQL Concepts Demonstrated

- Primary & Foreign Keys
- Constraints (NOT NULL, UNIQUE, CHECK)
- INNER, LEFT, RIGHT & FULL Joins
- Aggregate Functions
- Subqueries
- Stored Procedures
- Triggers
- Views
- Indexes

##Learning Outcomes

This project demonstrates practical experience in database design, SQL programming, data integrity, query optimization, inventory management, and sales reporting using Microsoft SQL Server.
